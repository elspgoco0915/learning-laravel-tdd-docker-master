# Learning TDD in Laravel

## Build the environment

Run the following command in docker directory.

```sh
$ docker compose build
```

## Run

Run the following command in docker directory.

```sh
$ make up
```

Open `http://localhost` in a browser.

## Stop

Run the following command in docker directory.

```sh
$ make stop
```
